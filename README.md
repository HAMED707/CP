## üöÄ Getting Started

Each folder contains `.cpp` solution files along with problem descriptions and approaches. To compile and run:

```bash
# Simple compilation
g++ -o solution solution.cpp

# With optimizations
g++ -O2 -o solution solution.cpp

# With modern C++ standard
g++ -std=c++17 -O2 -o solution solution.cpp

# Run
./solution
```
